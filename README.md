# Archetype Automation Engine

Archetype automation engine for data analysis, clustering, and reporting.

## Structure
- `src/`: Python pipeline code.
- `notebooks/`: Jupyter notebooks for exploration and validation.
- `data/`, `outputs/`: Data and results (gitignored).

## Setup
1. Clone: `git clone https://github.com/10anshika/archetype-automation-engine.git`
2. Install deps: `pip install -r requirements.txt` (create if needed).
3. Run: `python src/run_pipeline.py`

See `archetype_engine_docs.md` for details.
